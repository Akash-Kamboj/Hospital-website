hero-bg.jpg

  • Recommended size: 1920x1080 pixels
  • Format: JPG or PNG

2. About Page Background (about-bg.jpg)

about-bg.jpg

  • Hospital building or medical team image

Image Naming Convention

Please name your images exactly as specified above to ensure they work with the existing code.

Image Optimization Tips

  • Use image compression tools to reduce file size while maintaining quality
  • Consider using WebP format for better performance (with JPG/PNG fallbacks)
  • Ensure all images are properly licensed for use on your website
